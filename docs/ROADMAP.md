# Roadmap

This roadmap tracks the project phase by phase. Each phase should leave the repository in a runnable state.

## Phase Checklist

- [x] Phase 0: repo scaffold and documentation
- [x] Phase 1: multipurpose request planning and stub pipeline
- [ ] Phase 2: FastAPI service
- [x] Phase 3: LangGraph supervisor workflow
- [x] Phase 4: workflow routing and subgraphs
- [x] Phase 5: structured planning agent
- [x] Phase 6: backend routing agent
- [x] Phase 7: validation and retry logic
- [x] Phase 8: real model integration
- [x] Phase 9: polish and examples
- [x] Phase 10: optional MCP / Docker / CI / retrieval

## Notes

- Early phases should favor simple Python functions and local storage.
- Real model integrations come later, after the stub pipeline and API are stable.
- LangGraph should be introduced only when orchestration needs are clear.
- Phase 10 completed Docker, CI, local template retrieval, and local request caching.
